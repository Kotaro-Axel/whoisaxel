Hogar Universitario

Back-end Dev. Front-end Dev August 2021 - September 2021

Developed a centralized room-rental app for lessors who want to put their houses, apartments and rooms on rental and for those university students looking for a new home to move to.
It was developed under a one-month internship remote program with a four-member team, having weekly meetings to discuss objective definitions and progress feedback.
As a back-end dev role, the main project required a server-side application to manage requests and store consumer data. The server app was built as an API REST with PHP's Laravel as Main Framework, managing the data with MySQL.
As a front-end dev role, the main project requires a mobile application for the consumers to interact with. Built the mobile app with AngularJS's Ionic as a mobile oriented framework. tasks were focused on UI making and API Consuming.

VITRA

Devops Engineer. Back-end dev September 2022 - December 2022

Developed a travel-offer and vehicle-rental centralized application for companies or people who want to put their vehicles on rental or for those who want to offer travel transport to local destinations.
It was developed over a four-month deadline with a three-member team, having daily meetings to discuss objective definitions and progress feedback.
As Back-end dev role, the main project required a server-side application to manage requests and store consumer data. Developed the application under a micro-services oriented architecture. The full backend was built as three different API RESTS with JavaScript's NodeJS + Express as Main Framework, managing the data with MySQL and POSTGRESQL.
As Devops Engineer role, the project's back-end services and databases were required to be deployed to it remote access. Deployed the back-ends using micro-services strategy with AWS's EC2 Technology to hold the back-ends applications and AWS's RDS to manage the database instances.
